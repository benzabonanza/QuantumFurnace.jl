    2. Add eigenbasis_overlap_analysis function:
       ```
       function eigenbasis_overlap_analysis(
           L::Matrix{<:Complex},
           observables::Vector{<:Matrix{<:Complex}},
           observable_names::Vector{String},
           rho0::Matrix{<:Complex},
       ) -> OverlapAnalysisResult
       ```

       Implementation (from RESEARCH.md Pattern 3, verified against eigenmode_decomposition.jl):
       a. Full dense eigendecomposition: `F = eigen(L)`
       b. Sort by `abs.(real.(F.values))`: `perm = sortperm(abs.(real.(F.values)))`
       c. Apply permutation: `lambda = F.values[perm]`, `V = F.vectors[:, perm]`
       d. Initial state expansion: `alpha = V \ reshape(rho0, :)`
       e. For each observable O and mode k: `c[i,k] = dot(reshape(O, :), V[:, k]) * alpha[k]`
          - Use `conj(reshape(O, :))` dotted with `V[:, k]` because `dot(a,b) = a'*b` in Julia (conjugates first arg). Since observables are Hermitian, `vec(O)' * v_k` is what we want. Julia's `dot(vec(O), V[:,k])` computes `conj(vec(O))' * V[:,k]` which is wrong for Hermitian O. Instead use: `dot(V[:,k], reshape(O, :))` or explicit `sum(conj.(reshape(O,:)) .* V[:,k])` -- actually, the correct formula from the physics is `tr(O^dagger * R_k)` where `R_k = reshape(V[:,k], d, d)`. For Hermitian O, `O^dagger = O`. The vectorized form is `vec(O)^dagger * vec(R_k) = dot(reshape(O,:), V[:,k])`. In Julia, `dot(a,b) = conj(a)' * b = sum(conj.(a) .* b)`. So `dot(reshape(O,:), V[:,k])` computes `sum(conj(vec(O)) .* V[:,k])`. For Hermitian O in the eigenbasis (real diagonal for H, complex off-diagonal for correlations), this conjugates the observable vector. This IS the correct `vec(O)^H * v_k`.
          - So the correct call is: `o_k = dot(reshape(O, :), V[:, k])` and `c[i,k] = o_k * alpha[k]`
       f. Compute `exact_gap = abs(real(lambda[2]))`
       g. Compute `gap_mode_overlap[i] = abs(coeffs[i, 2])` (gap mode is index 2 in sorted eigenvalues)
       h. Compute `relative_gap_overlap[i] = abs(coeffs[i, 2]) / sum(abs.(coeffs[i, 2:end]))` (exclude steady state mode k=1)
          - Handle edge case: if denominator is zero (all coefficients zero), set to 0.0
