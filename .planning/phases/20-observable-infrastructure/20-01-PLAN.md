**Function 1: `build_total_magnetization(hamiltonian::HamHam, num_qubits::Int; trotter::Union{TrottTrott, Nothing}=nothing)`**

- Build M_z = sum(Z_i) / n in computational basis using `pad_term([Z], num_qubits, i)` for each site i in 1:num_qubits
- Accumulate into a pre-allocated `zeros(ComplexF64, dim, dim)` matrix using `.+=` (pad_term returns sparse -- use `Matrix{ComplexF64}(pad_term(...))` for conversion)
- Divide by `num_qubits` for per-site normalization (LOCKED decision: M_z = sum(Z_i) / n, NOT total sum)
- Select basis: `V = trotter !== nothing ? trotter.eigvecs : hamiltonian.eigvecs` (LOCKED decision: single function with trotter keyword, NOT separate _trotter function)
- Transform to eigenbasis: `Mz_eigen = Matrix{ComplexF64}(V' * Mz_comp * V)`
- Return `([Mz_eigen], ["Mz"])` tuple -- matching existing `(Vector{Matrix{ComplexF64}}, Vector{String})` pattern
- Add docstring following the existing `build_convergence_observables` docstring style, noting the per-site normalization and trotter keyword

**Function 2: `build_gap_estimation_observables(hamiltonian::HamHam, num_qubits::Int; trotter::Union{TrottTrott, Nothing}=nothing)`**

- Internally call `build_total_magnetization(hamiltonian, num_qubits; trotter=trotter)` to get M_z (DRY -- Claude's Discretion choice)
- Build H observable:
  - If `trotter !== nothing`: `H = Matrix{ComplexF64}(trotter.eigvecs' * hamiltonian.data * trotter.eigvecs)` (H is NOT diagonal in Trotter basis)
  - If `trotter === nothing`: `H = Matrix{ComplexF64}(diagm(ComplexF64.(hamiltonian.eigvals)))` (H IS diagonal in eigenbasis)
- Concatenate: `observables = [H; mz_obs]`, `names = ["H"; mz_names]` -- H first, then M_z
- Return tuple matching existing pattern
- Add docstring noting this returns the H + M_z gap estimation bundle (no ZZ correlations per user decision)
- IMPORTANT: Do NOT include ZZ correlations (DEFERRED decision)
