Create `test/test_dm_scaling.jl` with the following testsets.

Use the EnergyDomain (simplest non-trivial domain) with the existing 4-qubit TEST_* fixtures. The approach: use the Liouvillian matrix L directly for deterministic DM evolution, NOT `run_thermalization` (which is stochastic). Euler step: `rho(t+delta) = rho(t) + delta * reshape(L * vec(rho(t)), DIM, DIM)`.

**DMTST-03: Single-step error O(delta^2)**
- Build the Liouvillian: `config = make_liouv_config(EnergyDomain())`, `L = construct_lindbladian(TEST_JUMPS, config, TEST_HAM)`.
- Compute the exact one-step evolution via matrix exponential for reference: `rho_exact(delta) = reshape(exp(delta * L) * vec(rho0), DIM, DIM)`.
- Initial state: maximally mixed `rho0 = Matrix{ComplexF64}(I, DIM, DIM) / DIM`.
- Delta sweep: `deltas = [0.1, 0.05, 0.025, 0.0125]`.
- For each delta:
  - Euler step: `rho_euler = rho0 + delta * reshape(L * vec(rho0), DIM, DIM)`.
  - Exact step: `rho_exact = reshape(exp(delta * L) * vec(rho0), DIM, DIM)`.
  - Error: `norm(vec(rho_euler) - vec(rho_exact))` (Frobenius norm of vectorized difference).
- Compute ratios of consecutive errors: `errors[i] / errors[i+1]`. For O(delta^2), when delta halves the error should decrease by factor ~4 (ratio ~ 4).
- Test: each ratio is between 3.0 and 5.0 (allowing some tolerance for numerical effects).
- Print all errors and ratios for diagnostics.

**DMTST-04: Multi-step accumulated error O(delta)**
- Same Liouvillian L and initial state rho0 as above.
- Fixed total evolution time `T = 0.5`.
- Delta sweep: `deltas = [0.1, 0.05, 0.025, 0.0125]`.
- For each delta:
  - `num_steps = Int(round(T / delta))`.
  - Euler evolution loop: `rho = copy(rho0); for _ in 1:num_steps; rho .+= delta .* reshape(L * vec(rho), DIM, DIM); end`.
  - Exact evolution: `rho_exact_T = reshape(exp(T * L) * vec(rho0), DIM, DIM)`.
  - Error: `norm(vec(rho) - vec(rho_exact_T))`.
- Compute ratios. For O(delta), when delta halves the error should decrease by factor ~2 (ratio ~ 2).
- Test: each ratio is between 1.5 and 2.5.
- Print all errors and ratios for diagnostics.

**Note:** `exp(delta * L)` uses the dense matrix exponential from LinearAlgebra. For 256x256 this is fast. Compute `rho_exact_T = reshape(exp(T * L) * vec(rho0), DIM, DIM)` only once per testset (T is fixed).

**DMTST-05: Coherent term B consistency**

For a single jump operator (TEST_JUMPS[1], X on site 1), compute B via three methods and compare. Follow the pattern from B_test.jl but with proper assertions.

Compute B_bohr (exact, in Hamiltonian eigenbasis):
```julia
config_bohr = make_liouv_config(BohrDomain())
precomputed_bohr = precompute_data(BohrDomain(), config_bohr, TEST_HAM)
B_bohr = coherent_bohr(TEST_HAM, TEST_JUMPS[1], config_bohr)
rmul!(B_bohr, precomputed_bohr.gamma_norm_factor)
```

Compute B_time (time quadrature, in Hamiltonian eigenbasis):
```julia
config_time = make_liouv_config(TimeDomain())
precomputed_time = precompute_data(TimeDomain(), config_time, TEST_HAM)
B_time_val = B_time(TEST_JUMPS[1], TEST_HAM, precomputed_time.b_minus, precomputed_time.b_plus,
                    T0, BETA, SIGMA)
rmul!(B_time_val, precomputed_time.gamma_norm_factor)
```

Compute B_trotter (Trotter + time quadrature, in Trotter eigenbasis -- needs transformation):
```julia
config_trott = make_liouv_config(TrotterDomain())
precomputed_trott = precompute_data(TrotterDomain(), config_trott, TEST_TROTTER)
B_trott = B_trotter(TEST_JUMPS[1], TEST_TROTTER, precomputed_trott.b_minus, precomputed_trott.b_plus,
                     BETA, SIGMA)
rmul!(B_trott, precomputed_trott.gamma_norm_factor)
# Transform B_trotter from Trotter eigenbasis to Hamiltonian eigenbasis
B_trott_in_eigen = TEST_TROTTER.trafo_from_eigen_to_trotter' * B_trott * TEST_TROTTER.trafo_from_eigen_to_trotter
```

Tests:
- `@test norm(B_bohr - B_time_val) < TOL_QUADRATURE` (time quadrature error only)
- `@test norm(B_bohr - B_trott_in_eigen) >= norm(B_bohr - B_time_val) - 1e-10` (Trotter error >= time quadrature error, with small numerical margin)
- `@test norm(B_bohr - B_trott_in_eigen) < 0.1` (sanity bound: Trotter error should not be huge)
- Print all three norms for diagnostics.
