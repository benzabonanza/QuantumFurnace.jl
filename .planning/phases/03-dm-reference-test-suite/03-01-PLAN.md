**2. Create test/test_dm_detailed_balance.jl with two testsets:**

**DMTST-01: Bohr detailed balance** (3-qubit system)
- Create a LiouvConfig with `BohrDomain()`, `with_coherent=true`, `num_qubits=3`, and all other params matching the existing `make_liouv_config` pattern (BETA, SIGMA, a=BETA/30, b=0.4, NUM_ENERGY_BITS, W0, T0, NUM_TROTTER_STEPS_PER_T0).
- Call `construct_lindbladian(SMALL_JUMPS, config, SMALL_HAM)` to get the 64x64 Liouvillian.
- Use `eigen(liouv)` (NOT Arpack `eigs` -- matrix is small and dense).
- Find the eigenvalue with smallest `abs(real(eigenvalue))` -- this is the zero eigenvalue (fixed point).
- Extract its eigenvector, reshape to 8x8 DM, Hermitianize: `(dm + dm') / 2`, normalize: `dm ./= tr(dm)`.
- Test: `trace_distance_h(Hermitian(ss_dm), SMALL_GIBBS) < 1e-10`.

**DMTST-02: Domain error hierarchy** (4-qubit system, existing TEST_* fixtures)
- For each domain `[BohrDomain(), EnergyDomain(), TimeDomain(), TrotterDomain()]`:
  - Create config via `make_liouv_config(domain)` (existing factory, uses 4-qubit params).
  - For TrotterDomain, pass `trotter=TEST_TROTTER` to `construct_lindbladian`.
  - Call `construct_lindbladian(TEST_JUMPS, config, TEST_HAM; trotter=...)`.
  - Extract fixed point via `eigen` (256x256 matrix, fast enough).
  - Hermitianize and normalize.
  - Compute `trace_distance_h(Hermitian(ss_dm), TEST_GIBBS)`.
- Store distances in a Dict keyed by domain name symbol.
- Test hierarchy with tolerance: `@test distances[:bohr] <= distances[:energy] + 1e-12` (i.e., allow tiny numerical noise). Similarly for energy<=time and time<=trotter.
