**Helper functions (module-level, before testsets):**

1. `compare_krylov_dense(config_liouv, hamiltonian, jumps; trotter=nothing, krylovdim=30, howmany=4, tol=1e-10, n_dense_modes=4)`:
   - Build dense Lindbladian via `construct_lindbladian(jumps, config_liouv, hamiltonian; trotter=trotter)`
   - Extract dense reference via `extract_leading_eigendata(L_dense; n_modes=n_dense_modes)`
   - Run Krylov via `krylov_spectral_gap(config_liouv, hamiltonian, jumps; trotter=trotter, krylovdim=krylovdim, howmany=howmany, tol=tol)`
   - Return named tuple `(; krylov_result, dense_result, L_dense)`

2. `print_gap_summary(domain_name, balance_name, krylov_gap, dense_gap)`:
   - Print one-line summary: domain (15 chars), balance (5 chars), gap_krylov, gap_dense, abs error, relative error
   - Use `@printf` with scientific notation

3. `print_eigenvalue_table(krylov_eigs, dense_eigs; top_k=6)`:
   - Print formatted table header: idx | Re(krylov) | Re(dense) | abs_err | rel_err
   - Print top_k rows comparing eigenvalue real parts
   - Use `@printf` for alignment

4. `on_failure_diagnostics(krylov_result, dense_result)`:
   - Print "FAILURE DIAGNOSTICS:" header
   - Call `print_eigenvalue_table` with top_k=6
   - Print Krylov converged count, matvec_count, num_restarts

5. `run_le_convergence(domain, hamiltonian, jumps; deltas=[0.1, 0.01, 0.001], trotter=nothing, krylovdim=30, tol=1e-10)`:
   - Compute gap_L via Krylov Lindbladian path: `make_liouv_config(domain; with_coherent=true)` then `krylov_spectral_gap`
   - For each delta: create `make_thermalize_config(domain; with_coherent=true, delta=delta)`, run `krylov_spectral_gap`, compute error = abs(gap_L - gap_from_E)
   - Compute convergence orders from consecutive error pairs: `order = log(e_{i-1}/e_i) / log(d_{i-1}/d_i)`
   - Print formatted convergence table with header and separator
   - Return `(; gap_L, rows, orders)` where rows is vector of `(; delta, gap_from_E, error)` named tuples
