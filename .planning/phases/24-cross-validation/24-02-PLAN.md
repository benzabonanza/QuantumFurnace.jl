  <action>
Create `experiments/validate_gap_estimation.jl` following the `run_sweep.jl` pattern (standalone script with helper functions, Printf output, using QuantumFurnace).

**Usage:** `julia --project=. experiments/validate_gap_estimation.jl`

**Script structure:**

1. **Imports:** `using QuantumFurnace, LinearAlgebra, Printf, Dates`

2. **Constants (matching run_sweep.jl pattern but for TimeDomain):**
   - `DELTA = 0.01`
   - `BETA = 10.0`
   - `NUM_ENERGY_BITS = 12`
   - `W0 = 0.05`
   - `T0 = 2pi / (2^NUM_ENERGY_BITS * W0)`
   - `NUM_TROTTER_STEPS_PER_T0 = 10`
   - `SEED = 42`

3. **Helper: `build_heisenberg_hamiltonian(num_qubits, beta)`** -- Same as run_sweep.jl: uniform XXX chain with periodic=true, J=[1,1,1].

4. **Helper: `build_jumps(hamiltonian, num_qubits)`** -- Build single-site Pauli jump operators in Hamiltonian eigenbasis (NOT Trotter basis, because we use TimeDomain with_coherent=false). Pattern from research: pad_term, normalize by sqrt(n_jumps), transform to eigenbasis via `hamiltonian.eigvecs' * op * hamiltonian.eigvecs`. Same as run_sweep.jl but use `hamiltonian.eigvecs` instead of `trotter.eigvecs`.

5. **Helper: `make_matched_configs(num_qubits, beta, sigma)`** -- Create matching LiouvConfig and ThermalizeConfig with IDENTICAL physics parameters:
   - `domain = TimeDomain()`
   - `with_coherent = false` (per research recommendation)
   - `with_linear_combination = true`
   - `sigma = 1.0 / beta`
   - `a = beta / 30.0`
   - `b = 0.4`
   - `num_energy_bits, w0, t0, num_trotter_steps_per_t0` from constants
   - ThermalizeConfig additionally gets `mixing_time` and `delta`

6. **Helper: `extract_exact_result(L, hamiltonian, num_qubits)`** -- Build LindbladianResult from dense eigen:
   - `eig = eigen(L)`
   - `sorted_idx = sortperm(abs.(real.(eig.values)))`
   - ss_idx = sorted_idx[1], gap_idx = sorted_idx[2]
   - Extract steady state: `reshape(eig.vectors[:, ss_idx], dim, dim)`, `hermitianize!`, normalize by `tr()`
   - Extract gap mode: `reshape(eig.vectors[:, gap_idx], dim, dim)`
   - Return `LindbladianResult(liouvillian=L, fixed_point=ss_dm, gap_mode=gap_mode, spectral_gap=eig.values[gap_idx])`

7. **Main function: `validate_system(num_qubits; ntraj, total_time)`:**
   - Print header: "=== Validating n=$num_qubits ==="
   - Build Hamiltonian: `build_heisenberg_hamiltonian(num_qubits, BETA)`
   - Build jumps: `build_jumps(hamiltonian, num_qubits)`
   - Create configs: `make_matched_configs(num_qubits, BETA, 1.0/BETA)`
   - **Exact gap:** Print "Computing exact Liouvillian eigenvalues...", then `construct_lindbladian(jumps, liouv_config, hamiltonian)`, then `extract_exact_result(L, hamiltonian, num_qubits)`. Print exact eigenvalue and exact_gap = abs(real(...)).
   - **Trajectory gap:** Print "Running trajectory gap estimation (ntraj=$ntraj)...", build psi0 = [1, 0, ..., 0], call `estimate_spectral_gap(jumps, therm_config, psi0, hamiltonian; ntraj, save_every=10, seed=SEED, total_time, skip_initial=0.1)`. Print fitted gap, CI, best observable, R-squared.
   - **Cross-validate:** `cv = cross_validate_gap(estimated, exact_result)`. Print all CrossValidationResult fields.
   - **Pass/fail:** Print "PASS" if within_ci==true OR relative_error < 0.3, else "NEEDS MORE TRAJECTORIES".
   - Return cv

8. **`main()` function:**
   - Print sweep header with timestamp
   - Run n=4: `validate_system(4; ntraj=1000, total_time=50.0)`
   - Run n=6: `validate_system(6; ntraj=5000, total_time=100.0)`
   - Print summary: both results, total wall time
   - Print overall PASS/FAIL based on both results

9. **Call `main()` at the end of script.**

**Critical invariants:**
- Use `TimeDomain()` with `with_coherent=false` for BOTH LiouvConfig and ThermalizeConfig
- Use `hamiltonian.eigvecs` (not trotter.eigvecs) for jump operator basis transform (TimeDomain operates in Hamiltonian eigenbasis)
- sigma = 1.0 / beta (matching run_sweep.jl KMS convention)
- Use `construct_lindbladian` + dense `eigen()`, NOT `run_lindbladian` (which uses Arpack iterative solver)
- n=6 total_time=100.0 to capture enough decay curve (gap is O(0.01-0.1))
