Create `test/test_gns_trajectory.jl` with the following testsets. This file validates the entire GNS trajectory code path -- requirements GNS-01 and GNS-02.

**Import requirements:** The file uses fixtures from test_helpers.jl (SMALL_HAM, SMALL_JUMPS, SMALL_GIBBS, SMALL_DIM) and factory functions (make_small_liouv_config_gns, make_small_thermalize_config_gns). Also uses `using Random` for Xoshiro RNG seeding.

**Testset 1: "GNS-01: Lindbladian fixed point (EnergyDomain)"**
- Create `config = make_small_liouv_config_gns(EnergyDomain())`
- Construct Lindbladian: `liouv = construct_lindbladian(SMALL_JUMPS, config, SMALL_HAM)`
- Full eigendecomposition: `eig = eigen(liouv)` (64x64 dense matrix, same pattern as test_dm_detailed_balance.jl DMTST-01)
- Extract fixed point: `ss_idx = argmin(abs.(real.(eig.values)))`, reshape to 8x8, Hermitianize `(ss_dm + ss_dm') / 2`, normalize by `tr(ss_dm)`
- Validate fixed point is a valid DM: `@test isapprox(tr(ss_dm), 1.0, atol=1e-12)`, `@test isapprox(ss_dm, ss_dm', atol=1e-12)` (Hermitian), `@test all(eigvals(Hermitian(ss_dm)) .>= -1e-12)` (PSD)
- Compute GNS-to-Gibbs approximation gap: `gap = trace_distance_h(Hermitian(ss_dm), SMALL_GIBBS)`. Log it: `@info "GNS-01: GNS fixed point to Gibbs trace distance (approximation gap)" gap`. The gap should be nonzero (GNS does NOT reproduce exact Gibbs): `@test gap > 1e-6` (strictly positive). Also check it is bounded: `@test gap < 0.5` (sanity -- it should not be wildly far from Gibbs).

**Testset 2: "GNS-01: CPTP completeness (EnergyDomain)"**
- Create GNS thermalize config: `config = make_small_thermalize_config_gns(EnergyDomain(); delta=0.01)`
- Build trajectory framework: use the same pattern as test_cptp.jl. Construct `ham_or_trott = SMALL_HAM`, `precomputed_data = QuantumFurnace._precompute_data(config, ham_or_trott)`, `scratch = KrausScratch(ComplexF64, SMALL_DIM)`, then `fw = build_trajectoryframework(SMALL_JUMPS, ham_or_trott, config, precomputed_data, scratch, config.delta)`.
- For each jump operator index 1:length(SMALL_JUMPS), compute `K0'K0 + delta * R_j + U_res'U_res` and verify `isapprox(..., I(SMALL_DIM), atol=1e-10)`. Follow the exact same CPTP pattern from test_cptp.jl (fw.K0, fw.jumps[j], fw.residual_unitaries, fw.delta, fw.scaled_prefactor).
- NOTE: GNS has `with_coherent=false` so there is no U_B unitary -- the CPTP relation is K0'K0 + delta * sum(gamma_j * A_j' * A_j) = I where the channel is purely dissipative.

**Testset 3: "GNS-02: Trajectory convergence to GNS fixed point (EnergyDomain)"**
This is the key convergence test. Use `run_trajectories` with `ThermalizeConfigGNS`.
- First, compute the GNS reference fixed point (same as testset 1 but store it):
  ```julia
  liouv_config = make_small_liouv_config_gns(EnergyDomain())
  liouv = construct_lindbladian(SMALL_JUMPS, liouv_config, SMALL_HAM)
  eig = eigen(liouv)
  ss_idx = argmin(abs.(real.(eig.values)))
  ss_vec = eig.vectors[:, ss_idx]
  gns_fp = reshape(ss_vec, SMALL_DIM, SMALL_DIM)
  gns_fp = (gns_fp + gns_fp') / 2
  gns_fp ./= tr(gns_fp)
  ```
- Run trajectories: `config = make_small_thermalize_config_gns(EnergyDomain(); delta=0.01, mixing_time=5.0)`. Use `psi0 = zeros(ComplexF64, SMALL_DIM); psi0[1] = 1.0` (computational basis |0>). Run `result = run_trajectories(SMALL_JUMPS, config, psi0, SMALL_HAM; ntraj=1000, seed=42)`.
- Use delta=0.01 (delta_eff = 0.01 * 9 = 0.09, safely < 1.0) and ntraj=1000 for statistical convergence. mixing_time=5.0 gives 500 steps per trajectory.
- Convergence assertion: `dist = trace_distance_h(Hermitian(result.density_matrix), Hermitian(gns_fp))`. Log: `@info "GNS-02: Trajectory to GNS fixed point trace distance" dist`. Assert: `@test dist < 0.05`.
- DM validity on the final result ONLY (per user decision):
  - `@test isapprox(tr(result.density_matrix), 1.0, atol=1e-10)` (unit trace)
  - `@test isapprox(result.density_matrix, result.density_matrix', atol=1e-10)` (Hermitian)
  - `@test all(eigvals(Hermitian(result.density_matrix)) .>= -1e-10)` (PSD)
- Also log the GNS-to-Gibbs gap for Phase 18 baseline: `gibbs_dist = trace_distance_h(Hermitian(gns_fp), SMALL_GIBBS)`. `@info "GNS-02: GNS fixed point to Gibbs distance (Phase 18 baseline)" gibbs_dist`.
