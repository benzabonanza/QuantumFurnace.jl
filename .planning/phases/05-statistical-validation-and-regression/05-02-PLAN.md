  <files>test/reference/generate_references.jl, test/reference/energy_dm_reference.bson, test/reference/energy_traj_reference.bson, test/reference/trotter_coherent_dm_reference.bson, test/reference/trotter_coherent_traj_reference.bson</files>
  <action>
1. Create `test/reference/` directory.

2. Create `test/reference/generate_references.jl` with:
   - Header documenting purpose: one-time script to generate frozen BSON reference data. Run via `julia --project test/reference/generate_references.jl`.
   - Imports: `using QuantumFurnace, LinearAlgebra, Random, BSON`
   - Include test helpers: `include(joinpath(@__DIR__, "..", "test_helpers.jl"))`
   - Constants: `REF_DIR = @__DIR__`, `REF_DELTA = 0.1`, `REF_SEED = 12345`, `REF_NTRAJ = 1000`

   - Function `generate_dm_reference(domain; with_coherent::Bool, filename::String)`:
     * Build Liouvillian via `make_small_liouv_config(domain; with_coherent=with_coherent)` + `construct_lindbladian(SMALL_JUMPS, liouv_config, SMALL_HAM; trotter_kw...)`
     * For TrotterDomain: `trotter_kw = (; trotter=SMALL_TROTTER)`, otherwise `(;)`
     * Initial state: `psi0 = fill(ComplexF64(1.0), SMALL_DIM) / sqrt(SMALL_DIM)`, `rho0 = psi0 * psi0'`
     * Compute: `rho_dm = reshape(exp(REF_DELTA * L) * vec(rho0), SMALL_DIM, SMALL_DIM)`, Hermitianize: `(rho_dm + rho_dm') / 2`
     * Save as **plain Matrix{ComplexF64}** (NOT Hermitian wrapper): `BSON.bson(joinpath(REF_DIR, filename), Dict("rho" => Matrix(rho_dm), "delta" => REF_DELTA, "domain" => string(typeof(domain)), "with_coherent" => with_coherent, "num_qubits" => 3))`
     * Print confirmation: `println("Saved: $filename")`

   - Function `generate_traj_reference(domain; with_coherent::Bool, filename::String)`:
     * Build trajectory framework: `make_small_thermalize_config(domain; with_coherent, delta=REF_DELTA, mixing_time=Float64(REF_DELTA))`, `precompute_data`, `KrausScratch`, `build_trajectoryframework`
     * For TrotterDomain: use `SMALL_TROTTER` as ham_or_trott
     * Initial state: same psi0 as above
     * Seed: `Random.seed!(REF_SEED)`
     * Accumulate: loop `REF_NTRAJ` times, `step_along_trajectory!(psi, fw)`, accumulate `psi * psi'`
     * Average, Hermitianize: `(rho_traj + rho_traj') / 2`
     * Save as **plain Matrix{ComplexF64}**: `BSON.bson(joinpath(REF_DIR, filename), Dict("rho" => Matrix(rho_traj), "delta" => REF_DELTA, "domain" => string(typeof(domain)), "with_coherent" => with_coherent, "num_qubits" => 3, "seed" => REF_SEED, "ntraj" => REF_NTRAJ))`

   - Main body: call all 4 generators:
     * `generate_dm_reference(EnergyDomain(); with_coherent=false, filename="energy_dm_reference.bson")`
     * `generate_traj_reference(EnergyDomain(); with_coherent=false, filename="energy_traj_reference.bson")`
     * `generate_dm_reference(TrotterDomain(); with_coherent=true, filename="trotter_coherent_dm_reference.bson")`
     * `generate_traj_reference(TrotterDomain(); with_coherent=true, filename="trotter_coherent_traj_reference.bson")`

3. Run the generator: `julia --project test/reference/generate_references.jl`

4. Verify all 4 BSON files exist in `test/reference/`.

5. Quick round-trip sanity check: load each BSON file and verify the `rho` field is a `Matrix{ComplexF64}` of size `(SMALL_DIM, SMALL_DIM)`.

**Pitfall avoidance:**
- Always save `Matrix(rho)`, never `Hermitian(rho)` -- BSON may not round-trip Hermitian wrappers correctly
- Use `@__DIR__` for path resolution so the script works from any working directory
- delta=0.1 matches Phase 4 single-step parameters
- seed=12345 is distinct from Phase 4 test seeds (42, 123)
