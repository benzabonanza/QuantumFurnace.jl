**Testset: TVAL-06 - TrotterDomain coherent convergence to Gibbs state**

Implementation:

1. **Build Liouvillian** for TrotterDomain with_coherent=true using the 3-qubit system:
```julia
@testset "TVAL-06: TrotterDomain coherent convergence to Gibbs state" begin
    dim = SMALL_DIM  # 8
    delta = 0.01  # Claude's discretion: small enough for convergence, large enough for speed

    # Liouvillian for TrotterDomain + coherent
    liouv_config = make_small_liouv_config(TrotterDomain(); with_coherent=true)
    L = construct_lindbladian(SMALL_JUMPS, liouv_config, SMALL_HAM; trotter=SMALL_TROTTER)
    exp_L = exp(delta * L)  # Compute once, apply repeatedly

    # Gibbs state in Trotter eigenbasis (following DMTST-02 pattern)
    gibbs_comp = SMALL_HAM.eigvecs * SMALL_GIBBS * SMALL_HAM.eigvecs'
    gibbs_trott = Hermitian(SMALL_TROTTER.eigvecs' * gibbs_comp * SMALL_TROTTER.eigvecs)
```

2. **DM evolution via iterated matrix-vector multiply:**
```julia
    psi0 = fill(ComplexF64(1.0), dim) / sqrt(dim)
    rho_dm_vec = vec(psi0 * psi0')

    max_steps = 5000  # Safety cap
    dm_converged_step = 0
    for step in 1:max_steps
        rho_dm_vec = exp_L * rho_dm_vec
        rho_dm = reshape(rho_dm_vec, dim, dim)
        rho_dm = (rho_dm + rho_dm') / 2
        dist = QuantumFurnace.trace_distance_h(Hermitian(rho_dm), gibbs_trott)
        if dist < 1e-3
            dm_converged_step = step
            println("  DM converged at step $step (trace_dist=$(round(dist; sigdigits=4)))")
            break
        end
    end
    @test dm_converged_step > 0  # DM must converge within max_steps
```

3. **Trajectory evolution** for the same number of steps using `run_trajectories`:
```julia
    # Run trajectory batch for the same total time
    total_time = dm_converged_step * delta
    therm_config = make_small_thermalize_config(TrotterDomain();
        with_coherent=true, delta=delta, mixing_time=total_time)

    Random.seed!(42)
    result = run_trajectories(SMALL_JUMPS, therm_config, psi0, SMALL_HAM;
        trotter=SMALL_TROTTER, total_time=total_time, delta=delta, ntraj=10_000)
    rho_traj = result.rho_mean
```

If `run_trajectories` prints verbose output (validate_config!, print_press), suppress with redirect_stdout or just let it print -- it is acceptable for a standalone test script.

4. **Assertions:**

```julia
    # Both DM and trajectory must be within 1e-3 of Gibbs

end
```

**Critical implementation details:**
- Use `delta = 0.01` (Claude's discretion: matches TEST_DELTA, small enough for DM convergence).
- `max_steps = 5000` safety cap. At delta=0.01, this is 50 time units.
- Compute `exp_L = exp(delta * L)` ONCE, then iterate `rho_dm_vec = exp_L * rho_dm_vec` per step (Pitfall 6 from research).
- Gibbs state in Trotter eigenbasis: `gibbs_comp = ham.eigvecs * gibbs * ham.eigvecs'` then `gibbs_trott = trott.eigvecs' * gibbs_comp * trott.eigvecs` (Pitfall 4 from research, same pattern as DMTST-02).
- Use `Random.seed!(42)` before `run_trajectories` (Pitfall 1 from research).
- 10,000 trajectories per user decision.
- `run_trajectories` already Hermitianizes rho_mean internally, so no need to post-process.
- The test runs `run_trajectories` with the full multi-step evolution internally (the function handles stepping). No manual per-step trajectory loop needed.
- If `run_trajectories` verbose output is annoying, it is still acceptable -- this is a standalone test file.
