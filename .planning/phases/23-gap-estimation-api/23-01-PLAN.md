1. **SpectralGapResult struct** (all concrete types, no type parameter needed -- avoids Aqua issues):
   ```julia
   struct SpectralGapResult
       gap::Float64                          # gap from best-fit observable
       gap_ci::Tuple{Float64, Float64}       # 95% CI on gap
       gap_se::Float64                       # standard error on gap
       best_observable::String               # name of best observable
       best_r_squared::Float64               # R-squared of best fit
       per_observable::Vector{FitResult}     # all fits, one per observable
       observable_names::Vector{String}      # names matching per_observable order
       ntraj::Int                            # number of trajectories
       total_time::Float64                   # simulation total time
       save_every::Int                       # save interval in steps
       seed::Int                             # RNG seed used
       skip_initial::Float64                 # fraction of data skipped for fitting
   end
   ```

2. **_select_best_observable(fits, names)** internal helper:
   - Iterate fits, collect indices where `fit.converged && fit.gap > 0.0`
   - Among valid fits, select the one with highest `fit.r_squared`
   - Fallback: if no valid fit, use `argmax(fit.r_squared for fit in fits)` (may have gap=0 or be unconverged)
   - Return `(best_idx, best_name, best_r_squared)`

3. **estimate_spectral_gap function** with signature:
   ```julia
   function estimate_spectral_gap(
       jumps::Vector{JumpOp},
       config::AbstractThermalizeConfig,
       psi0::Vector{<:Complex},
       hamiltonian::HamHam;
       observables::Union{Nothing, Vector{<:Matrix{<:Complex}}} = nothing,
       observable_names::Union{Nothing, Vector{String}} = nothing,
       ntraj::Int = 1000,
       save_every::Int = 10,
       total_time::Real = config.mixing_time,
       delta::Real = config.delta,
       seed::Union{Int, Nothing} = nothing,
       trotter::Union{TrottTrott, Nothing} = nothing,
       skip_initial::Float64 = 0.0,
   )
   ```
   Implementation steps:
   - If `observables === nothing`, call `build_gap_estimation_observables(hamiltonian, config.num_qubits; trotter=trotter)` to get default [H, Mz] bundle. CRITICAL: forward `trotter` to BOTH this call and `run_observable_trajectories` (Pitfall 6 from research).
   - Validate: if `observables` provided but `observable_names` is nothing, throw `ArgumentError`. Validate lengths match.
   - Call `run_observable_trajectories(jumps, config, psi0, hamiltonian; observables=observables, save_every=save_every, ntraj=ntraj, total_time=total_time, delta=delta, seed=seed, trotter=trotter)`.
   - For each observable `i`, extract time series via `traj_result.measurements_mean[i, :]` (row = observable, column = time -- Pitfall 4). Cast to Float64 with `Float64.(...)`.
   - Call `fit_exponential_decay(traj_result.times, obs_series; skip_initial=skip_initial)` for each.
   - Call `_select_best_observable(fits, observable_names)`.
   - Construct and return `SpectralGapResult` with all fields populated. Use `traj_result.seed` (the actual seed used), `Float64(total_time)`, and the passed `skip_initial`.
