_exp_decay_model(t, p) = @. p[1] * exp(-p[2] * t) + p[3]

`FitResult` struct with these fields (all Float64 or obvious types):
- `gap::Float64` -- Fitted decay rate (spectral gap estimate), constrained > 0
- `amplitude::Float64` -- Fitted amplitude A
- `offset::Float64` -- Fitted offset C (asymptotic value)
- `gap_ci::Tuple{Float64, Float64}` -- 95% confidence interval on gap (lower, upper)
- `gap_se::Float64` -- Standard error on gap
- `r_squared::Float64` -- Goodness of fit (1 - RSS/TSS), NOT clamped (can be negative for bad fits)
- `converged::Bool` -- Whether LM optimization converged
- `residuals::Vector{Float64}` -- Fit residuals
- `times_used::Vector{Float64}` -- Time points used (after skip_initial)
- `values_used::Vector{Float64}` -- Data values used (after skip_initial)

Add a docstring to `FitResult` matching the project's documentation style (see `ConvergenceData` in structs.jl for reference).

Internal helper `_log_linear_initial_guess(times::AbstractVector{<:Real}, values::AbstractVector{<:Real}) -> Vector{Float64}`:
1. Estimate plateau C from last 20% of data: `tail_start = max(1, n - div(n, 5)); C_guess = sum(values[tail_start:n]) / (n - tail_start + 1)`
2. Shift data: `y_shifted = values .- C_guess`
3. Filter positive values: `mask = y_shifted .> 1e-10`
4. If fewer than 3 valid points, use fallback: `A = values[1] - values[end]`, `gap = 1.0 / (times[end] - times[1])`, `C = values[end]`
5. Otherwise: linear regression on `log.(y_shifted[mask])` vs `times[mask]` using `X \ log_y` (least-squares solve). Extract `A = exp(coeffs[1])`, `gap = max(-coeffs[2], 1e-6)`.
6. Return `[A_guess, gap_guess, C_guess]`

Internal helper `_compute_r_squared(values::AbstractVector{<:Real}, residuals::AbstractVector{<:Real}) -> Float64`:
- `ss_res = sum(residuals .^ 2)`
- `y_mean = sum(values) / length(values)`
- `ss_tot = sum((values .- y_mean) .^ 2)`
- Return `1.0 - ss_res / ss_tot` (do NOT clamp -- negative R-squared is valid diagnostic)

Main function `fit_exponential_decay(times::AbstractVector{<:Real}, values::AbstractVector{<:Real}; skip_initial::Float64=0.0, p0::Union{Nothing, Vector{Float64}}=nothing, level::Float64=0.95) -> FitResult`:
1. Validate inputs: `length(times) == length(values)`, `length(times) >= 4` (need at least 4 points for 3-parameter fit), `0.0 <= skip_initial < 1.0`
2. Apply skip_initial BEFORE fitting (critical -- see RESEARCH pitfall 4): `start_idx = max(1, floor(Int, skip_initial * length(times)) + 1)`, then slice `times_fit = times[start_idx:end]`, `values_fit = values[start_idx:end]`
3. Validate at least 4 points remain after truncation
4. Generate initial guess: if `p0 === nothing`, call `_log_linear_initial_guess(times_fit, values_fit)`, else use `p0`
5. Set bounds: `lower = [-Inf, 0.0, -Inf]` (gap > 0), `upper = [Inf, Inf, Inf]`
6. Call `curve_fit(_exp_decay_model, Float64.(times_fit), Float64.(values_fit), Float64.(p0_used); lower=lower, upper=upper)` -- CRITICAL: do NOT pass a weight vector (see RESEARCH pitfall 6 about corrupted covariance with unit weights)
7. Extract results using StatsAPI names (NOT deprecated names):
   - `params = coef(fit)` -> `[A, gap, C]`
   - `se = stderror(fit)` -> gap_se = `se[_IDX_GAP]`
   - `ci = confint(fit; level=level)` -> gap_ci = `ci[_IDX_GAP]` (this is a Tuple{Float64,Float64})
   - `resid = residuals(fit)` (Vector{Float64})
   - `converged = fit.converged`
8. Compute R-squared: `r2 = _compute_r_squared(Float64.(values_fit), resid)`
9. Return `FitResult(gap, A, C, gap_ci, gap_se, r2, converged, resid, Float64.(times_fit), Float64.(values_fit))`

- Arguments section (times, values)
- Keyword arguments section (skip_initial, p0, level)
- Returns section (FitResult)
