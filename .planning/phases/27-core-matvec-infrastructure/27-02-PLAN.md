The test file structure:

using Test

**Testset 1: "KrylovWorkspace construction"**
- Create workspace for EnergyDomain KMS: `ws = KrylovWorkspace(config_kms, TEST_HAM, TEST_JUMPS)`
- Verify `ws.B_total !== nothing` (KMS with_coherent=true)
- Verify all scratch matrices are `DIM x DIM`
- Create workspace for EnergyDomain GNS: `ws_gns = KrylovWorkspace(config_gns, TEST_HAM, TEST_JUMPS)`
- Verify `ws_gns.B_total === nothing` (GNS has no coherent)

**Testset 2: "Round-trip: matvec vs dense (EnergyDomain KMS, no coherent)"**
- Config: `make_liouv_config(EnergyDomain(); with_coherent=false)`
- Build dense reference: `L_dense = construct_lindbladian(TEST_JUMPS, config, TEST_HAM)`
- Build workspace: `ws = KrylovWorkspace(config, TEST_HAM, TEST_JUMPS)`
- For 10 random density matrices (use `random_density_matrix(NUM_QUBITS)`):
  - Dense: `v_dense = L_dense * vec(Matrix(rho))` (random_density_matrix returns Hermitian, wrap in Matrix)
  - Matvec: `L_rho = apply_lindbladian!(ws, Matrix(rho), config, TEST_HAM)`
  - `@test norm(v_dense - vec(L_rho)) < 1e-12`

**Testset 3: "Round-trip: matvec vs dense (EnergyDomain KMS, with coherent)"** [MATVEC-05 + MATVEC-09]
- Config: `make_liouv_config(EnergyDomain(); with_coherent=true)`
- Same pattern as testset 2. This validates the coherent term `-i[B, rho]` is correctly applied.
- 10 random density matrices, tolerance < 1e-12

**Testset 4: "Round-trip: matvec vs dense (EnergyDomain GNS)"** [MATVEC-01 GNS]
- Config: `make_liouv_config_gns(EnergyDomain())` (new factory from task 1)
- Same pattern as testset 2. Validates GNS transition function + silently skipped coherent.
- 10 random density matrices, tolerance < 1e-12

**Testset 5: "Round-trip: adjoint matvec vs dense adjoint (EnergyDomain KMS)"** [MATVEC-07 + MATVEC-09]
- Config: `make_liouv_config(EnergyDomain(); with_coherent=true)` (test with coherent to verify sign flip)
- Build dense reference: `L_dense = construct_lindbladian(TEST_JUMPS, config, TEST_HAM)`
- Dense adjoint is `L_dense'` (Julia's adjoint = conjugate transpose of the superoperator matrix)
- Build workspace: `ws = KrylovWorkspace(config, TEST_HAM, TEST_JUMPS)`
- For 10 random density matrices:
  - Dense adjoint: `v_adj_dense = L_dense' * vec(Matrix(rho))`
  - Matvec adjoint: `L_adj_rho = apply_adjoint_lindbladian!(ws, Matrix(rho), config, TEST_HAM)`
  - `@test norm(v_adj_dense - vec(L_adj_rho)) < 1e-12`

**Testset 6: "Adjoint duality check: tr(X' * L(Y)) == tr(L*(X)' * Y)"**
- Additional mathematical validation of adjointness (Hilbert-Schmidt inner product)
- Config: KMS with coherent
- For 5 pairs of random matrices X, Y:
  - `lhs = tr(Matrix(X)' * apply_lindbladian!(ws, Matrix(Y), config, TEST_HAM))`
  - (re-zero ws.rho_out is handled by apply_lindbladian! itself)
  - `rhs = tr(apply_adjoint_lindbladian!(ws, Matrix(X), config, TEST_HAM)' * Matrix(Y))`
  - `@test abs(lhs - rhs) < 1e-11` (slightly looser due to two matvec calls)

**Testset 7: "Zero allocations in matvec hot path"** [MATVEC-08]
- Config: `make_liouv_config(EnergyDomain(); with_coherent=true)`
- Build workspace, create one random density matrix as `Matrix{ComplexF64}`
- Warmup call: `apply_lindbladian!(ws, rho, config, TEST_HAM)`
- Measure: `allocs = @allocated apply_lindbladian!(ws, rho, config, TEST_HAM)`
- `@test allocs == 0`
- Same for adjoint:
- Warmup: `apply_adjoint_lindbladian!(ws, rho, config, TEST_HAM)`
- Measure: `allocs_adj = @allocated apply_adjoint_lindbladian!(ws, rho, config, TEST_HAM)`
- `@test allocs_adj == 0`

IMPORTANT: `random_density_matrix(NUM_QUBITS)` returns a `Hermitian` wrapper. The matvec functions take `AbstractMatrix`, but `Hermitian` may cause allocation issues with `mul!`. Convert to `Matrix` before passing: `rho = Matrix(random_density_matrix(NUM_QUBITS))`.

Also add the test file to `test/runtests.jl` -- find the pattern of how other test files are included and add:
