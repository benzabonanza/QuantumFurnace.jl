  <action>
    Create src/mixing.jl with the following components:

    1. **MixingTimeEstimate struct** (immutable, following FitResult pattern):
       ```julia
       struct MixingTimeEstimate
           # Fit parameters (from FitResult)
           fitted_gap::Float64
           amplitude::Float64
           offset::Float64
           gap_ci::Tuple{Float64, Float64}
           gap_se::Float64
           r_squared::Float64
           converged::Bool
           # Mixing time results
           mixing_time::Float64                           # primary answer
           mixing_time_extrapolated::Union{Nothing, Float64}
           mixing_time_actual::Union{Nothing, Float64}
           target_epsilon::Union{Nothing, Float64}
           # Full fit for advanced users
           fit_result::FitResult
       end
       ```

    2. **estimate_mixing_time function**:
       ```julia
       function estimate_mixing_time(
           result::ThermalizeResults;
           skip_initial::Float64 = 0.2,
           target_epsilon::Union{Nothing, Float64} = nothing,
           extrapolate::Bool = false,
           level::Float64 = 0.95,
       )
       ```
       Implementation:
       - Extract `times = result.time_steps` and `dists = result.trace_distances`
       - Validate: `length(times) >= 10` or throw ArgumentError
       - If `extrapolate == true && target_epsilon === nothing`, throw ArgumentError("target_epsilon required when extrapolate=true")
       - Call `fit_exponential_decay(Float64.(times), Float64.(dists); skip_initial=skip_initial, level=level)` -- pass skip_initial THROUGH, do NOT apply it first (per Research pitfall 3, option b)
       - Call `_check_fit_quality(fit, target_epsilon)` for quality gate warnings
       - Compute `t_mix_actual = _find_actual_mixing_time(times, dists, target_epsilon)`
       - Compute `t_mix_extrap = extrapolate ? _extrapolate_mixing_time(fit, target_epsilon) : nothing`
       - Primary `mixing_time`: if `extrapolate`, use `t_mix_extrap` (or `NaN` if extrapolation failed); if not extrapolate and target_epsilon provided, use `t_mix_actual` (or `NaN` if not reached); if no target_epsilon, use `last(times)` (total simulation time)
       - Return MixingTimeEstimate with all fields populated

    3. **_check_fit_quality helper** (internal, uses @warn):
       - Warn if `fit.r_squared < 0.95`: "Fit R-squared = {value} < 0.95. Single-exponential model may not describe the data well."
       - Warn if `target_epsilon !== nothing && fit.offset > 0.1 * target_epsilon`: "Fit offset C = {value} is large relative to target epsilon = {target_epsilon}. Extrapolation may be unreliable."
       - Warn if `!fit.converged`: "Levenberg-Marquardt optimization did not converge. Fit results are unreliable."
       - Warn if `fit.gap_se > 0.5 * fit.gap && isfinite(fit.gap_se)`: "Gap standard error ({gap_se}) exceeds 50% of fitted gap ({gap}). Confidence interval is very wide."

    4. **_find_actual_mixing_time helper**:
       - If target_epsilon === nothing, return nothing
       - Iterate dists: find first index where `dists[i] <= target_epsilon`, return `times[i]`
       - If not found, return nothing

    5. **_extrapolate_mixing_time helper**:
       - Guards: return nothing if target_epsilon === nothing, fit.gap <= 0, fit.amplitude <= 0
       - `effective_target = target_epsilon - fit.offset`
       - Guard: return nothing if `effective_target <= 0` (offset exceeds target)
       - Guard: return nothing if `effective_target >= fit.amplitude` (already below target at t=0)
       - Return `-log(effective_target / fit.amplitude) / fit.gap`

    6. **Update src/QuantumFurnace.jl**:
       - Add `include("mixing.jl")` AFTER `include("fitting.jl")`
       - Add exports: `export estimate_mixing_time, MixingTimeEstimate`

    Add a file header comment block matching the codebase style (see fitting.jl header as template):

    ```
    # ============================================================================
    ```

    Add docstrings to MixingTimeEstimate, estimate_mixing_time, following the style in fitting.jl (field descriptions for struct, Arguments/Keyword Arguments/Returns/Example for function).
