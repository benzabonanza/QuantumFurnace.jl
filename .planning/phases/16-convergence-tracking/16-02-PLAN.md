**Testset 8: "run_trajectories_convergence integration" (CONV-01, CONV-02, CONV-03)**
This is the critical integration test. Use the 4-qubit test system with EnergyDomain.

Setup:
```julia
config = make_thermalize_config(EnergyDomain(); with_coherent=true, delta=0.01, mixing_time=5.0)
observables, names = build_convergence_observables(TEST_HAM, NUM_QUBITS)
obs_gibbs = QuantumFurnace._compute_gibbs_observable_values(TEST_GIBBS, observables)
```

Run:
```julia
traj_result, conv_data = run_trajectories_convergence(
    TEST_JUMPS, config, TEST_HAM.eigvecs[:, 1], TEST_HAM;
    gibbs=TEST_GIBBS,
    observables=observables,
    observable_names=names,
    batch_size=200,
    n_batches=5,
    seed=42,
)
```
Use batch_size=200, n_batches=5 (total=1000 trajectories). Use `psi0 = TEST_HAM.eigvecs[:, 1]` (ground state in eigenbasis = first column of identity = [1,0,...,0] in eigenbasis, but safer to use actual eigenvector column).

Wait -- in eigenbasis, the state vector is represented in the eigenbasis. So `psi0` should be a vector in the eigenbasis. The simplest choice is the ground state: `psi0 = zeros(ComplexF64, DIM); psi0[1] = 1.0` (first eigenvector in eigenbasis representation).

Assertions:
- `conv_data isa ConvergenceData`
- `length(conv_data.batch_sizes) == 5`
- `all(conv_data.batch_sizes .== 200)`
- `conv_data.cumulative_n_traj == [200, 400, 600, 800, 1000]`
- `length(conv_data.trace_distances) == 5`
- `all(isfinite, conv_data.trace_distances)` and `all(conv_data.trace_distances .>= 0)`
- CONV-01: Trace distance at last batch < trace distance at first batch (convergence). Use generous tolerance -- just check `conv_data.trace_distances[end] < conv_data.trace_distances[1]`.
- `size(conv_data.observable_values) == (NUM_QUBITS + 1, 5)` (5 obs, 5 batches)
- `conv_data.observable_names == names`
- `length(conv_data.observable_gibbs_values) == NUM_QUBITS + 1`
- CONV-02: The energy observable (last row of observable_values) at the final batch is closer to the Gibbs value than at the first batch: `abs(conv_data.observable_values[end, end] - obs_gibbs[end]) < abs(conv_data.observable_values[end, 1] - obs_gibbs[end])`.
- CONV-03 check is implicit in CONV-02 (same mechanism for H).
- `traj_result isa TrajectoryResult`
- `traj_result.n_trajectories == 1000`
- `traj_result.seed == 42`
- `size(traj_result.rho_mean) == (DIM, DIM)`
- Verify rho_mean is approximately a valid density matrix: `isapprox(tr(traj_result.rho_mean), 1.0; atol=1e-6)` and approximately Hermitian.
