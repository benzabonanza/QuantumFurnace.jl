**1. Imports and constants:**
```julia
using QuantumFurnace
```

Define shared constants:

- `WINDOW_SIZE = 3` (discretion: compare windows of 3 batches)
- `MIN_BATCHES = 5` (discretion: burn-in before convergence checking)
- `NUM_ENERGY_BITS = 12`
- `W0 = 0.05`
- `T0 = 2pi / (2^NUM_ENERGY_BITS * W0)`
- `NUM_TROTTER_STEPS_PER_T0 = 10`
- `SEED = 42` (discretion: deterministic, reproducible)

**2. Helper: `build_heisenberg_xxx(num_qubits::Int, beta::Float64)`**
- Builds uniform 1D Heisenberg XXX chain with periodic boundaries, J=1.0
- `terms = Vector{Vector{Matrix{ComplexF64}}}([[X, X], [Y, Y], [Z, Z]])`
- `coeffs = [1.0, 1.0, 1.0]`
- Returns `HamHam(terms, coeffs, num_qubits, beta; periodic=true)`

**3. Helper: `build_trotter_system(hamiltonian::HamHam, num_qubits::Int)`**
- Creates `TrottTrott(hamiltonian, T0, NUM_TROTTER_STEPS_PER_T0)`
- Builds jump operators: single-site Paulis (X, Y, Z) on each site
- Jump normalization: `sqrt(3 * num_qubits)` (3 Paulis x num_qubits sites)
- For each Pauli on each site:
  - `op = Matrix(pad_term([pauli], num_qubits, site)) ./ norm_factor`
  - `in_eigen = hamiltonian.eigvecs' * op * hamiltonian.eigvecs`
  - `orthogonal = (op == transpose(op))`
  - `herm = (op == op')`
  - `push!(jumps, JumpOp(op, in_eigen, orthogonal, herm))`
- Returns `(trotter, jumps)` -- typed as `Vector{JumpOp{Matrix{ComplexF64}}}`

**4. Helper: `run_experiment(; kwargs...)`**
- Accepts: `jumps`, `hamiltonian`, `trotter`, `num_qubits`, `beta`, `sigma`, `delta`, `db_type::Symbol` (:KMS or :GNS), `mixing_time`, `n_max`, `seed`, `output_path`, `label`
- Constructs config:
  - KMS: `ThermalizeConfig(num_qubits=num_qubits, with_coherent=true, with_linear_combination=true, domain=TrotterDomain(), beta=beta, sigma=sigma, a=beta/30.0, b=0.4, num_energy_bits=NUM_ENERGY_BITS, w0=W0, t0=T0, num_trotter_steps_per_t0=NUM_TROTTER_STEPS_PER_T0, mixing_time=mixing_time, delta=delta)`
  - GNS: `ThermalizeConfigGNS(num_qubits=num_qubits, with_coherent=false, with_linear_combination=true, domain=TrotterDomain(), beta=beta, sigma=sigma, a=beta/30.0, b=0.4, num_energy_bits=NUM_ENERGY_BITS, w0=W0, t0=T0, num_trotter_steps_per_t0=NUM_TROTTER_STEPS_PER_T0, mixing_time=mixing_time, delta=delta)`
- Builds Gibbs reference: `QuantumFurnace._gibbs_in_trotter_basis(hamiltonian, trotter)` -- CRITICAL: must be Trotter basis, not energy basis (Quick-20 fix)
- Builds observables: `build_convergence_observables_trotter(hamiltonian, trotter, num_qubits)`
- Creates initial state: `psi0 = zeros(ComplexF64, 2^num_qubits); psi0[1] = 1.0`
- Calls `run_trajectories_adaptive(jumps, config, psi0, hamiltonian; gibbs=gibbs_trotter, observables=observables, observable_names=obs_names, batch_size=BATCH_SIZE, n_max=n_max, convergence_threshold=CONVERGENCE_THRESHOLD, patience=PATIENCE, min_batches=MIN_BATCHES, window_size=WINDOW_SIZE, seed=seed, trotter=trotter, total_time=mixing_time, delta=delta)`
- Wraps result: `ExperimentResult(config, traj_result, QuantumFurnace._extract_hamiltonian_params(hamiltonian), metadata)` with metadata from `QuantumFurnace._capture_metadata(; wall_time_seconds=wall_time, extra=Dict{Symbol,Any}(:convergence => QuantumFurnace._convergence_to_dict(conv_data), :sigma_rule => sigma_rule_string, :db_type => string(db_type), :label => label))`
- Saves via `save_experiment(result, output_path)`
- Returns `(result, conv_data, wall_time)`

**5. Main sweep loop:**
- Parameter grid: `system_sizes = [4, 6, 8]`, `betas = [5.0, 10.0, 20.0]`
- Mixing time: `mixing_time = 2.0 * beta` (discretion: scales with beta for sufficient mixing at high beta)
- Output directory: detect project root via `dirname(dirname(@__FILE__))` (since script is in `experiments/`), then `joinpath(project_root, "experiments")`. Use `mkpath` to ensure it exists.
- `failed_experiments = String[]`
- Loop over `n in system_sizes`, `beta in betas`:
  - Build Hamiltonian once: `build_heisenberg_xxx(n, beta)`
  - Build Trotter system once: `build_trotter_system(hamiltonian, n)`
  - Run 3 experiments per (n, beta):
    1. KMS at sigma=1/beta: filename `kms_n$(n)_beta$(Int(beta)).bson`
    2. GNS at sigma=1/beta: filename `gns_sigma1_n$(n)_beta$(Int(beta)).bson`
    3. GNS at sigma=0.5/beta: filename `gns_sigma05_n$(n)_beta$(Int(beta)).bson`
  - Each experiment wrapped in try/catch: on success print convergence status + trace distance + wall time; on failure push label to `failed_experiments` and continue
- Progress logging: one line per experiment start with timestamp, one line per experiment completion with status (CONVERGED or HIT CAP), final trace distance, trajectory count, and wall time
- At end: print total wall time and list of failed experiments (if any)

**6. Update `.gitignore`:**
Add `/experiments/` entry (locked decision: experiments/ is gitignored).
